# presidents.fyi

A ranked, opinionated list of every US president up to George HW Bush. Live at [presidents.fyi](https://presidents.fyi).

Built with [Astro](https://astro.build) + React. The ranking data lives in a Google Sheet and is fetched at build time.

## Local development

```sh
npm install
```

Create a `.env` file at the repo root with a Google Sheets API key (read access to the public spreadsheet is enough):

```
GOOGLE_API_KEY=your_key_here
```

Then:

```sh
npm run dev      # http://localhost:4321
npm run build    # static output to ./dist
npm run preview  # serve the build locally
```

## How it works

- **Data** — [src/data/presidents.ts](src/data/presidents.ts) fetches the `Tiers` sheet from Google Sheets at build time. Edit the sheet, rebuild, and the site picks up the changes.
- **Home page** ([src/pages/index.astro](src/pages/index.astro)) — renders the tier list. Clicking a president opens a drawer ([src/components/TierList.tsx](src/components/TierList.tsx)) and pushes a `/president/N` URL via `history.pushState`, so the URL bar matches what's on screen.
- **Per-president pages** ([src/pages/president/[number].astro](src/pages/president/[number].astro)) — pre-rendered at build time, one per president. These are what search engines and direct-link visitors see. Each has its own title, description, OG image, and `ProfilePage`/`Person` JSON-LD.
- **Sitemap** — generated by `@astrojs/sitemap` (see [astro.config.mjs](astro.config.mjs)).
